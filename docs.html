安装

多种方式安装 Metro UI，选择最适合你项目的方式

方式一：通过 npm 安装（推荐）

# 使用 npm npm install metro-ui-design-system # 使用 yarn yarn add metro-ui-design-system # 使用 pnpm pnpm add metro-ui-design-system

方式二：通过 CDN 引入

<!-- 在 HTML 中添加 CSS --> <link rel="stylesheet" href="https://unpkg.com/metro-ui-design-system/dist/metro-ui.min.css"> <!-- 在 body 结束前添加 JavaScript --> <script src="https://unpkg.com/metro-ui-design-system/dist/metro-ui.min.js"></script>

方式三：手动下载

从 GitHub Releases 下载最新版本，将 dist/metro-ui.css 和 dist/metro-ui.js 复制到你的项目中。

快速开始

按照以下步骤，5 分钟内创建你的第一个 Metro UI 应用

  1. 创建 HTML 文件
    <!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Metro App</title> <link rel="stylesheet" href="node_modules/metro-ui-design-system/dist/metro-ui.min.css"> </head> <body> <!-- 你的内容 --> <script src="node_modules/metro-ui-design-system/dist/metro-ui.min.js"></script> </body> </html>
  2. 添加 Live Tile
    <!-- 翻转磁贴 --> <div class="tile tile-medium" data-tile="flip"> <div class="tile-face tile-face-front"> <div class="tile-icon">📊</div> <div class="tile-title">仪表板</div> </div> <div class="tile-face tile-face-back"> <div class="tile-content">查看统计</div> </div> </div>
  3. 添加自定义下拉框
    <!-- 自定义选择器 --> <div class="metro-select"> <div class="metro-select-trigger"> <span class="placeholder">选择选项</span> <svg class="metro-select-arrow">...</svg> </div> <div class="metro-select-panel"> <div class="metro-option" data-value="1">选项 1</div> <div class="metro-option" data-value="2">选项 2</div> </div> </div>
  4. 初始化组件
    // 在页面加载后初始化 <script> document.addEventListener('DOMContentLoaded', function() { MetroUI.init(); }); </script>

API 参考

详细的 JavaScript API 文档

MetroUI.init()

初始化所有 Metro UI 组件

MetroUI.init(); // 自动初始化页面上的所有组件
MetroUI.toast(options)

显示提示消息

参数 类型 默认值 说明
type string 'info' 提示类型：success / error / warning / info
message string - 提示消息内容
duration number 3000 显示时长（毫秒）
position string 'top-right' 位置：top-right / top-left / bottom-right / bottom-left
MetroUI.toast({ type: 'success', message: '保存成功！', duration: 5000, position: 'top-right' });
MetroUI.modal.open(id) / MetroUI.modal.close(id)

打开或关闭模态框

// 打开模态框 MetroUI.modal.open('my-modal'); // 关闭模态框 MetroUI.modal.close('my-modal');

主题定制

通过 CSS 变量自定义主题颜色

<style> :root { /* 主题颜色 */ --metro-accent: #0078d4; --metro-success: #107c10; --metro-warning: #ff8c00; --metro-error: #d13438; /* 间距 */ --metro-space-1: 4px; --metro-space-2: 8px; --metro-space-3: 12px; --metro-space-4: 16px; /* 字体 */ --metro-font-family: 'Segoe UI', system-ui, sans-serif; --metro-fs-base: 14px; } /* 暗色主题 */ [data-theme="dark"] { --metro-bg: #1e1e1e; --metro-fg: #ffffff; --metro-line: #3f3f3f; } </style>
