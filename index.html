安装与引入Installation

Metro UI 是纯 HTML/CSS/JS 实现，零依赖。可通过 npm、CDN 或手动下载引入。Metro UI is pure HTML/CSS/JS with zero dependencies. Install via npm, CDN, or manual download.

方式一：npmOption 1: npm
npm install metro-ui-design-system
方式二：CDNOption 2: CDN
<link rel="stylesheet" href="https://unpkg.com/metro-ui-design-system/dist/metro-ui.min.css"> <script src="https://unpkg.com/metro-ui-design-system/dist/metro-ui.min.js"></script>
方式三：本地文件Option 3: Local Files
<link rel="stylesheet" href="styles/tokens.css"> <link rel="stylesheet" href="styles/base.css"> <link rel="stylesheet" href="styles/tiles.css"> <link rel="stylesheet" href="styles/components.css"> <link rel="stylesheet" href="styles/layout.css"> <link rel="stylesheet" href="styles/motion.css"> <script src="scripts/metro.js"></script>
